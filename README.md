### `world-of-tanks`
**The  web version of `World of Tanks`**

### Bootup
```bash
# yarn chrome 💐
yarn start
```

### Keyboard shortcuts
- Shot: `Blankspace`
- Move: `↑ ↓ ← →`
- Pause/Continue: `Enter`

### Timeline
2016/12 - 2017/3

### Mode
- The lair section has been changed to an `iron wall`.
- The firing mode is `single shot`·
